Getting Started and Keyboard Tips

Screen reader mode

  • JAWS users: Turn off Virtual Cursor with JAWS Key + Z.
  • NVDA and Narrator users: Use Focus Mode.
  • The Lab is optimized for keyboard interaction while the screen reader is in Focus Mode or Browse Mode is off.

Starting a session

  1. Select a learning mode.
  2. Select one or more WCAG principles.
  3. Select the number of questions, or Sprint time when using Sprint mode.
  4. Activate Start Lab Session.

Answering questions

  • The question is read automatically.
  • Press Q to repeat the question.
  • Press A to read the answer choices.
  • Use 1 through 4, Tab, or form-field navigation to move through answer buttons.
  • Activate an answer with Space or Enter.
  • After answering, Continue To Next Question receives focus.
  • Press C, Space, or Enter to continue.

Reviewing missed questions

  • After a completed session, activate Review Missed Questions.
  • Alt+M opens Review Missed Questions efficiently when Focus Mode is active or Browse Mode is off.
  • Each review item is announced automatically.
  • Activate Next Missed Question to continue.
  • On the final review item, activate Finish Review.
  • After review is complete, Start New Session receives focus.

Optional shortcuts

  • Alt+S - Start Lab Session
  • Alt+M - Review Missed Questions
  • Alt+T - Open Statistics
  • Alt+H - Return Home
  • Q - Repeat the current question
  • A - Read answer choices
  • C - Continue after answering
  • N - Skip question
  • R - Read remaining Sprint time
  • Escape - Collapse this Getting Started section when it is expanded
