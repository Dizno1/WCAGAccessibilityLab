Accessibility Lab

Thank you for testing the WCAG Accessibility Lab.

GETTING STARTED

1. Download the ZIP file.
2. Extract all files to a folder.
3. Open the extracted folder.
4. Activate "StartAccessibilityLab.bat".
5. The lab will open in your default web browser.

REQUIREMENTS

- Windows 10 or Windows 11
- Chrome, Edge, Firefox, or another modern browser
- No installation required

NOTES

- Always extract the ZIP before running the launcher.
- Saved Challenge items remain on your computer until exported or cleared.

Thank you for helping test the WCAG  Accessibility Lab.
